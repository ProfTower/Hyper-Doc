## **Common Errors**

- The `submit` event happens on the `form` element, even though the `button` can trigger the `submit` event. Therefore, we should add an event listener to the `form` element (not the `button` nor `input` element). Once the event listener is added to the `form` element, the `event.preventDefault()` works as expected.
